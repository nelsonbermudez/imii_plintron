Información Importante

Para cancelar un registro negativo necesita:

  • El IMEI exacto del dispositivo reportado
  • La fecha y hora exacta del reporte original (formato: AAAAMMDDHHMMSS)
  • Una justificación válida para la cancelación
Ingrese exactamente 15 dígitos numéricos
Formato: AAAAMMDDHHMMSS (Ejemplo: 20241025143000 = 25/Oct/2024 14:30:00)
